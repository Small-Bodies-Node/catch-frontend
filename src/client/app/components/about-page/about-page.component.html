ABOUT

CATCH is a moving target search tool for large data sets, such as time-domain or near-Earth object surveys. For a given target, CATCH identifies images that cover the ephemeris position. Use CATCH to help identify pre-discovery detections, to examine past cometary activity, or to understand why a survey did not detect or report observations of an object. CATCH has an interactive web-based interface and a fully featured API for end user's applications.

CATCH limitations and caveats

CATCH does not perform detection-based searches, i.e., it cannot determine if an object was observed. Instead, it identifies images that could have observed a target, based the ephemeris. This search requires that the ephemeris correctly describes the target's motion on the sky. CATCH properly considers the source data observatory when computing ephemerides. CATCH does not consider ephemeris uncertainties in the search. This capability has been reserved for a future enhancement.

Data sources

Near-Earth Asteroid Tracking (NEAT) survey data is from the NASA Planetary Data System archive of Bauer and Lawrence (2013):

  • NEAT GEODSS, Haleakala Observatory, April 1996 – May 1998.
  • NEAT Palomar Tri-Cam, Palomar Observatory, November 2001 &ndash March 2003.
